Project Overview: Data Bending with Audacity

This second piece in the series explores a more complex data bending technique using Audacity to manipulate the visual data of an image, producing controlled digital artifacts.

Category:

Audio Software Data Manipulation

Tools Used:

Audacity, Preview

File Format:

TIF (Tagged Image File Format)

Process Description

I first imported my image as a .tif file into Audacity. Then I copied sections of the data and applied the reverse effect to enhance the wave patterns of the distortion. Finally, I exported the modified file and opened it in Preview to observe the glitch effects.
