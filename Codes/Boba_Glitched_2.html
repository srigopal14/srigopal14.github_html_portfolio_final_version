Boba Glitched 2: Audacity Data Bending

Project Overview

This second piece in the series explores a more complex data bending technique using Audacity to manipulate the visual data of an image, producing controlled digital artifacts.

Process & Execution

  • Imported the image as a .tif file into Audacity.
  • Copied and manipulated sections of the data, applying the reverse effect for distortion.
  • Exported and opened the image in Preview to observe the resulting glitch effects.

Reflections

This project continues the exploration of digital errors and creativity, showing how controlled data manipulation can create visually striking outcomes.
