Project Overview: Data Bending with TextEdit

This project is the first in an experimental digital art series, focusing on glitch art created by manually manipulating a file's code, also called data bending.

Category:

Data Manipulation / Glitch Art

Tools Used:

TextEdit (Mac), Preview

File Source:

JPEG Image

Process Description

I used TextEdit on Mac to open the original JPEG image. Then I copied and pasted sections after ~20% into the file to avoid corrupting the header. Finally, I opened the modified file in Preview to observe the glitch effects produced by this manipulation.
