Project Overview

This project served as a comprehensive digital portfolio, built entirely within GitHub, showcasing a wide array of artwork and presentations created over the past years of college.

Process & Execution

  1. Created a 10-page website containing portfolio content, artwork, and presentations from college projects.
  2. Imported all videos and images using GitHub code structure for proper organization and deployment.
  3. Designed clear headers linking to specific reviews for easy navigation across pages.
  4. Tested the entire portfolio locally and via GitHub Pages to ensure seamless viewing.
  5. Reflected on the process to improve workflow for future portfolio projects.

Reflections

This project allowed me to demonstrate my skills in both graphic design and code, while creating a cohesive portfolio experience. It emphasized structured presentation and highlighted my evolution as a digital artist.
