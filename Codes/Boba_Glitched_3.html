Project Overview

This final piece in the Boba Glitched series moves beyond pure data bending into complex layer and filter manipulation in Adobe Photoshop, achieving intentional digital breakdown aesthetics.

Process & Execution

  1. Imported the image into Photoshop and cleared the background to black.
  2. Copied and pasted the image twice after masking off the first layer.
  3. For the first copy, unchecked the Green and Red channels (leaving only Blue).
  4. For the second copy, unchecked Green and Blue channels (leaving only Red).
  5. Super-imposed the color layers using arrow keys to create RGB separation glitch.
  6. Applied Halftone Pattern (Filter Gallery → Sketch → Line, Size: 2, Contrast: 5).
  7. Adjusted curves via Adjustment Layer to modify filter effects.
  8. Converted Curves layer into a Smart Object.
  9. Used Lasso Tool to copy/paste sections and move them to create vertical distortion and final glitch effect.

Reflections

This project pushes the exploration of digital errors and creative manipulation, demonstrating how layered Photoshop techniques can transform imagery into visually compelling glitch art.
